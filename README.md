# Proyecto Futbol - Django


## Funcionalidades

- CRUD de Equipos
- CRUD de Jugadores
- CRUD de Noticias
- Buscador de jugadores
- Panel de administración
- Página About

## Cómo ejecutar el proyecto

1. Clonar el repositorio
2. Crear entorno virtual
3. Instalar Django
4. Ejecutar migraciones
5. Ejecutar servidor

python manage.py runserver
